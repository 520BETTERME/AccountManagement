由于自己注册的各个平台的账号实在太多了，脑子不好使有点记不住，便用java写了个小
工具用来管理这些账号和密码，原理很简单，就是用dom4j解析xml文件，利用加密与解密
算法达到目的。
-----------------------------------------------------------------------
使用方法：
1、按照要求输入密码和安全问题及答案，重启程序；
2、输入刚才设置的密码登录；
3、输入平台的账户信息，其中平台，用户名及密码是必须的，剩下的可选的，输完信
   息后，按“保存”按钮保存账户信息；
4、搜索栏可输入平台名来搜索对应的账户信息；
5、右击放大镜图标可以显示出所有保存的账户信息；
6、右边的退出图标是删除所有账户信息及登录信息的，如果删除后，下次使用将需要重新
   设置登录信息；
7、修改账号信息需要输入与待修改账户相同平台及用户名的账号信息,程序会覆盖原来的,
   暂时没有实现删除账号的功能。
-----------------------------------------------------------------------
注意事项：
1、sources文件夹千万不要动！！！
2、没有安装JDK 1.8的同学，可以直接通过exe启动本程序，文件夹下的setup.bat、AccountManagement.jar文件可以直接删除，jre文
件夹千万不要动，否则极大可能导致程序无法启动；
3、安装了JDK1.8的同学，可以直接通过setup.bat启动本程序,jre文件夹及exe
文件可以删除，如果想运行exe文件，必须得保留jre文件夹；
4、没有安装JDK1.8，但是安装了JDK1.8以下的版本，由于没有测试，不知道能否可以通过
bat文件启动本程序。
-----------------------------------------------------------------------
项目是用idea+maven写的。
